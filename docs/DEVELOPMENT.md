# Руководство по запуску в режим разработки

## Установка зависимостей

```bash
npm install
```

## Запуск dev-сервера

```bash
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:5173`

Dev-сервер автоматически перезагружает страницу при изменении файлов.

## Проверка типов

```bash
npm run check-types
```

## Линтинг

```bash
npm run lint
```
